# git 提交代码流程

## 1. 同步git仓库代码

```git
# 同步仓库代码到本地仓库
git pull
```

## 2. 检查同步结果

### 若本地代码和仓库代码有冲突， 则按照下面流程解决

#### 1. 暂存自己的改动

```git
git stash
```

#### 2. 同步更新获取最新的改动

```
git pull
```

#### 3. 恢复本地的改动

```
git stash pop
```

#### 4. 在代码中解决冲突

### 3. 查看状态

```
git status
```

### 4. 同步git仓库代码

```
git pull
```

### 5. 选择要提交的文件

```
git add ./要提交文件的路径, 没路径默认提交当前目录下所有需要提交的内容
```

### 6. 提交文件, 并添加提交内容备注

```
git commit -m '要添加的备注内容'
```

### 7. 提交代码到git仓库

```
git push
```
